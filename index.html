Get Started Key Concepts

Build your static site using just the fundamental building blocks of the web!

You use HTML, CSS and JavaScript to prototype and implement your design, which makes Trio the perfect tool for those who love to design in the browser.

Trio's use of common metaphors for its project assets, its approach to page composition, its organized collections of metadata and its tag-based JavaScript callback mechanism serve to create a powerful but easy to learn and use platform upon which to create your static sites.

Trio doesn't require a templating engine or a front-end framework for extending your composites. Instead, Trio exposes its tag-based JavaScript callback mechanism.

Include the data-trio-callback attribute in any HTML tag and Trio will call the JavaScript module that it names, passing it the tag, and the metadata that Trio has collected.

Extend Your Pages With ES6+ JavaScript!

In fragment file source/fragments/products.html:

<!--
template: products.html
title: Products By Category
appendToTarget: true
data: productsByCategory.json
-->
<h1 class="product-list-title">Product List By Category</h1>
<ul data-trio-callback="listAllProductsByCategory" class="product-list">
    <li class="product-in-stock product-in-stock-heading">
        <span>Category</span>
        <span>Name</span>
        <span>Price <span class="product-in-stock-heading--smaller">(US$)</span></span>
    </li>
</ul>

In tag-based JavaScript file source/callbacks/listAllProductsByCategory.js:

module.exports = ({$tag, site}) =>
    site.dataCatalog.productsByCategory
        .filter(prod => prod.inStock)
        .forEach(prod => $tag.append(`
            <li class="product-in-stock">
                <span>${prod.category}</span>
                <span>${prod.name}</span>
                <span>${prod.price.toFixed(2)}</span>
            </li>
        `));
