Get Started

Install Node and Trio:

  • Step 1.

    Install Node

    If you don't already have Node installed, please go to nodejs.org and follow the installation instructions you find there.

  • Step 2.

    Install Trio

    To install Trio, run the following from the command line:

    npm i -g @4awpawz/trio
