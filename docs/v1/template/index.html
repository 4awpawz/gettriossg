Template

A template is a required asset that is associated with one or more fragment assets (i.e. multiple fragments can associate with a single template). The association between a template and a fragment is made by the fragment in its front matter using the predefined "template" front matter property and assigning it the name of a template. When composing a web page, Trio merges the fragment's content within the template's tag that has the data-trio-fragment attribute. The combined content of a single template and a single fragment forms the basis of a composite, which Trio saves as a web page in your project's root/public folder.

  • file location: root/source/templates
  • file type and content: .html and markup
  • applicable data-trio-* attributes: data-trio-fragment, data-trio-include, data-trio-callback, data-trio-data, data-trio-link
