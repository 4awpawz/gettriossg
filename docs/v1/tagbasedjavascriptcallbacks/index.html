Tag-Based JavaScript Callbacks

Tag-based JavaScript callbacks are Node modules that are located in your project's root/source/callbacks folder, which Trio calls asynchronously using async/await when it encounters an HTML tag decorated with the data-trio-callback attribute.

Each callback module must export a function which, when called by Trio, is passed a single object from which it can destructure any or all of the following properties:

  • $tag: a Cheerio-wrapped object around the tag that declared the callback
  • $: a Cheerio-wrapped object around the composite that contains the tag
  • asset: the metadata for the fragment associated with the composite
  • site: all the metadata that Trio has accumulated for the composite
  • Cheerio: an instance of Cheerio that can be used to create new Cheerio wrappers around objects should the need arise
