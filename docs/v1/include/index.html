Include

An include is an optional asset that is inserted into fragment and template assets.

When composing a page, Trio merges the fragment's content with the tag in the template that is decorated with the data-trio-fragment attribute. The merging of content from a single template with a single fragment, followed by the merging of any content from any include assets that either asset may declare, forms the basis of a composite, which Trio saves as a page.

To insert an include asset, you decorate the tag that you are targeting as the insertion point with a data-trio-include attribute, and assign it the name of the include file (or a reference to a front matter property to be found in the fragment asset whose value is the name of the file) as the attributes value.

  • file location: root/source/includes
  • file type/content: .html/HTML or .md/Markdown
  • front matter required: no
  • required predefined front matter properties: none
  • optional predefined front matter properties: appendToTarget
  • applicable data-trio-* attributes: data-trio-callback, data-trio-link
