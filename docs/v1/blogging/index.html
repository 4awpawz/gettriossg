Blogging

Trio was built from the ground up to be used as a blogging platform. Trio supports the creation of modern-day blogs with the features that your readers have come to expect, such as categories and tags, and Trio's structured blog-specific metadata and its tag-based JavaScript callback mechanism makes implementing modern blogs a snap.

A sample blog project that incorporates categories, tags, intricate page linking and also demonstrates Trio's many numerous features in general is available at 4awpawz/trio-blog-example and is viewable at https://4awpawz.github.io/trio-blog-example-pages.

Please feel free to clone the sample blog project and then build it, run it and examine its code.
